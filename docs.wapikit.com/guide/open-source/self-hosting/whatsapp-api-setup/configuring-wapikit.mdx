---
title: Configuring WapiKit
description: Learn how to configure the wapikit for your Whatsapp Business Account
---


To receive and send messages, you need to configure Whatsapp Business Account in WapiKit.


<Steps>
    <Step>
        <Accordion title="Visit Your WapiKit Application Settings">
            <Note>Visit WapiKit Dashboard and click on the settings icon in the left sidebar or navigate to  `/settings?tab=whatsapp-business-account`.</Note>
                <img src="/assets/updating-business-account-details/settings-button.png" />
        </Accordion>
    </Step>
    <Step>
        <Accordion title="Copy paste the API key">
            <Frame>
                <img src="/assets/updating-business-account-details/business-account-api-settings.png" />
            </Frame>
        </Accordion>
    </Step>
    <Step>
        <Accordion title="Save Configuration">
            <Note>Click on update to save the whatsapp API configurations</Note>
        
            <Frame>
                <img src="/assets/updating-business-account-details/update-button.png" />
            </Frame>
        </Accordion>
    </Step>
</Steps>
